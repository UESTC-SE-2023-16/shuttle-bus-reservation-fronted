## 开发

### 安装工具

1. 安装 [pnpm](https://pnpm.io/zh/installation#使用独立脚本安装) - 包管理器
2. 使用 `pnpm env use --global 20` 安装 Node.js v20.10.0

### 安装依赖

```bash
pnpm i
```

### 启动开发服务器

```bash
pnpm dev
```

> [!NOTE]
> 此时前端服务器已启动，会在命令行中显示前端服务器的地址  
> 请确保后端服务器已启动  

> [!WARNING]
> 如果想预览生产环境的效果，请看[这里](#预览生产环境)
